# Breast Cancer Data Analysis

This project performs **Exploratory Data Analysis (EDA)** on the Breast Cancer dataset to understand the characteristics that differentiate malignant and benign tumors.

## Project Overview

The goal of this project is to explore tumor-related features and identify patterns that may help in cancer diagnosis. Various data visualization techniques are used to analyze the relationships between features and the diagnosis variable.

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Analysis Steps

* Data loading and exploration
* Feature distribution analysis
* Correlation analysis
* Data visualization
* Insights and conclusions

## Key Insights

* Malignant tumors tend to have larger **radius** and **perimeter** values.
* Features such as **radius_mean**, **perimeter_mean**, and **area_mean** show strong correlations.
* Visualizations reveal clear differences between malignant and benign tumors.

## Dataset

Breast Cancer Wisconsin Dataset.

## Conclusion

The analysis reveals important patterns in tumor characteristics and shows that several features may be useful for building machine learning models for breast cancer classification.
